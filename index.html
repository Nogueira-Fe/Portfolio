Procurando um desenvolvedor FullStack?

Felipe Nogueira O olhar e paixão que uma criança tem ao olhar para o mundo, é o mesmo que o meu ao aprender novas tecnologias.

Graduando em Análise e Desenvolvimento de Sistemas. Ao longo da minha jornada acadêmica e autodidata, adquiri conhecimentos em diversas tecnologias essenciais para o desenvolvimento web e backend. Iniciei meus estudos com HTML e CSS, onde aprendi a estruturar e estilizar páginas da web, e em seguida, evoluí para JavaScript, que me permitiu adicionar interatividade e dinamismo aos projetos. Aprofundei-me em TypeScript para melhorar a qualidade do meu código e trabalhar com tipagem estática, o que levou à minha familiarização com React, uma biblioteca poderosa para construir interfaces de usuário eficientes e escaláveis. Além disso, desenvolvi um entendimento sólido de Bancos de Dados e SQL, fundamentais para a gestão e manipulação de dados de forma eficaz. Mais recentemente, tenho focado em Node.js, explorando o desenvolvimento backend para criar aplicações completas e robustas. Estou sempre em busca de novos desafios e oportunidades para aplicar e expandir meus conhecimentos.

Este projeto foi desenvolvido utilizando apenas HTML/CSS, tendo como proposta aprender a desenvolver uma página HTML, estilizar e resolver possíveis problemas por conta própria!

Extremamente similar ao projeto de HTML/CSS, porém, com algumas melhorias devido à novas experiências e aprendizados. Desenvolvido utilizando React, que envolveu novas funcionalidades para fazer a "réplica" do primeiro projeto!
